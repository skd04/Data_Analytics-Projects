## PROJECT 1
# COVID-19 Data Analysis using SQL Server

## Project Overview

This project explores global COVID-19 data using SQL Server.  
The goal is to analyze infection trends, death rates, and vaccination progress across countries and continents.

---

### Dataset

Tables used:

- CovidDeaths
- CovidVaccinations

These contain daily country-level data including population, cases, deaths, and vaccinations.

---

### Key Questions Answered

1. What is the death rate among confirmed cases?
2. What percentage of population was infected?
3. Which countries had the highest infection rates?
4. Which countries had the highest death counts?
5. How did vaccination progress over time?
6. What percentage of population has been vaccinated?

---

### SQL Concepts Used

This project demonstrates practical use of:

- SELECT and filtering
- ORDER BY
- Aggregate functions (SUM, MAX, AVG)
- GROUP BY
- CAST and data type conversion
- JOINS
- Window functions (OVER, PARTITION BY)
- Running totals
- CTE (Common Table Expression)
- Temporary tables
- Views

---

### Key Analysis Highlights

#### Death Percentage
Calculated likelihood of dying after infection for a specific country.

#### Infection Rate
Measured total cases relative to population size.

#### Highest Infection Countries
Identified countries with the largest outbreaks.

#### Vaccination Progress
Used running totals to track vaccination rollout over time.

#### Population Vaccinated
Calculated percentage of population vaccinated using cumulative data.
