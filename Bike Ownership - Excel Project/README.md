# Bike Ownership Analysis - EXCEL

## Project Overview

This project focuses on analyzing a bike ownership dataset to understand customer behavior and identify patterns that influence bike purchases.

The goal of this project is to practice data cleaning, analysis, and visualization, and to present insights in a simple and clear way.

---

## Objectives

- Clean and prepare raw data for analysis  
- Explore customer demographics  
- Identify factors affecting bike purchases  
- Create a visual dashboard for better understanding  

---

## Dataset

The dataset contains customer-related information such as:

- Age  
- Gender  
- Income  
- Marital Status  
- Education  
- Occupation  
- Commute Distance  
- Region  
- Bike Purchase Status  

---

## Data Cleaning

The following steps were performed:

- Checked for missing values  
- Standardized column formats  
- Fixed inconsistent values  
- Renamed columns for better readability  
- Removed unnecessary data  

---

## Data Analysis

Analysis was done to understand:

- Which age group buys bikes more  
- Income impact on bike purchases  
- Gender-based trends  
- Effect of commute distance  
- Relationship between occupation and bike ownership  

---

## Dashboard

A dashboard was created to visualize key insights from the data.

You can find the dashboard image in the **output** folder.

---

## Key Insights

- Middle-aged customers are more likely to purchase bikes  
- Higher income groups show more interest in buying bikes  
- Commute distance plays an important role in decision-making  
- Certain occupations have higher bike ownership rates  

---

- Excel (Data Cleaning and Dashboard)  
- SQL (Data Analysis)
