\# Airbnb Data Analysis Dashboard



\## Project Overview



This project focuses on analyzing Airbnb listing data to understand pricing patterns, location trends, and factors that influence rental prices.



The main goal of this project is to practice data analysis and visualization by working with a real dataset and building an interactive dashboard.



\---



\## Dataset



The dataset contains Airbnb listing details such as:



\- Price  

\- Zipcode  

\- Number of Bedrooms  

\- Number of Bathrooms  

\- Location  

\- Listing details  



\---



\## Data Cleaning



Basic data cleaning steps were performed before analysis:



\- Checked for missing values  

\- Cleaned inconsistent data  

\- Adjusted data types where required  

\- Prepared dataset for visualization  



\---



\## Dashboard



Below is a snapshot of the dashboard created for this project:



!\[Airbnb Dashboard](output/dashboard.png)



The dashboard includes:

\- Profit trends over time  

\- Total price comparison by zipcode  

\- Average price based on number of bedrooms  



\---



\## Key Insights



\- Certain zipcodes have significantly higher listing prices  

\- Listings with more bedrooms tend to have higher average prices  

\- Price trends show variation across months  

\- Location plays a major role in determining listing value
